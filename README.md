# excel-projects

📊 Vrinda Store Sales Dashboard (Excel Project)
📌 Project Overview

This project presents an interactive sales dashboard for Vrinda Store using Microsoft Excel.
The dashboard helps analyze sales performance, customer behavior, and order trends for the year 2022.

It provides meaningful insights through data visualization and helps in better business decision-making.

🎯 Objectives
Analyze sales vs orders trends
Understand customer demographics (age & gender)
Compare men vs women purchasing behavior
Track order status (delivered, cancelled, returned)
Identify top-performing states
Analyze sales channels performance
📂 Dataset Details

The dataset includes:

Order ID
Customer details
Gender & Age group
Sales amount
Order status
State
Sales channel
📊 Dashboard Features
🔹 Sales vs Orders Analysis
Monthly comparison of total sales and number of orders
Helps identify seasonal trends
🔹 Gender-based Analysis
Comparison between men vs women purchases
Identifies target customer segment
🔹 Order Status Insights
Delivered vs Cancelled vs Returned orders
Helps track operational efficiency
🔹 State-wise Sales
Top contributing states in revenue
Useful for regional strategy planning
🔹 Age & Gender Segmentation
Customer distribution across age groups
Helps in targeted marketing
🔹 Sales Channels
Performance comparison of different platforms (Amazon, Flipkart, etc.)
🛠 Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Data Cleaning
Dashboard Design
📈 Key Insights
Women contribute more to total sales compared to men
Majority of orders are successfully delivered
Certain states dominate overall sales
Online channels like Amazon/Flipkart drive most revenue
Adult age group is the primary customer segment
🚀 How to Use
